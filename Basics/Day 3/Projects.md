# 🐍 Python Practice Projects

This is a curated list of **Python practice projects** for all levels – Beginner → Medium → Hard → Advanced.  
Each project focuses on different Python concepts like variables, input/output, conditional expressions, loops, string manipulation, logical operators, and more.  

---

## **Level 1 – Beginner Projects (Basics)**  
_Concepts: variables, input, typecasting, if-else, conditional expressions, string basics_

1. **Positive/Negative & Even/Odd Checker**  
   - Input a number, print if it’s positive/negative and even/odd.  
   - Concepts: `if-else`, `modulus (%)`, `conditional expression`.

2. **Maximum & Minimum Finder**  
   - Input 2 or 3 numbers, find max and min.  
   - Concepts: `conditional expression`, `typecasting`.

3. **Age & Status Checker**  
   - Input age, print “Adult” if ≥18 else “Not Adult”.  

4. **Simple Greeting Program**  
   - Input name and age, greet differently if minor/adult.  
   - Concepts: input, if-else, string formatting.

5. **Simple Calculator – Two Numbers**  
   - Input two numbers, perform +, -, *, /.  

6. **Temperature Converter**  
   - Convert Celsius → Fahrenheit and Fahrenheit → Celsius.  
   - Concepts: variables, arithmetic operations, input/output.

7. **Even Numbers Printer**  
   - Input a range, print all even numbers in that range.  
   - Concepts: loops, modulus.

## **Level 2 – Medium Projects**  
_Concepts: string methods, logical operators, loops, multiple conditions, indexing_

8. **Credit Card Masker**  
   - Show only last 4 digits, mask others with X.  

9. **Name Formatter**  
   - Input full name: capitalize first letters, uppercase/lowercase, count spaces.  

10. **Phone Number Cleaner**  
    - Count dashes, replace with spaces, optional formatting to XXX-XXX-XXXX.  

11. **Weather Alert System**  
    - Input temperature and weather conditions; print messages.  
    - Concepts: logical operators, if-elif-else.

12. **Password Strength Checker (Medium)**  
    - Length ≥8, contains number and alphabet.  

13. **Simple Interest Calculator**  
    - Input principal, rate, time → calculate simple interest.  

14. **Number Reverser**  
    - Input a number, print its reverse.  
    - Concepts: string/number conversion, slicing.  

15. **Character Counter**  
    - Input string, count vowels, consonants, digits, special characters.  

16. **Multiplication Table Generator**  
    - Input number, print multiplication table.  

17. **Word Counter in a Sentence**  
    - Input sentence, count words.  

18. **Even/Odd Numbers in Range**  
    - Input start & end, print lists of even and odd numbers separately.

23. **Fibonacci Series Generator**

24. **Factorial Calculator**

25. **Prime Number Checker**

26. **Temperature Analysis**

27. **Simple Banking System**

28. **Shopping Cart Total Calculator**

## ✅ Practice Instructions

1. Start with **Level 1**, solve **all projects yourself**, no copy-paste.  
2. Move to **Level 2**, focus on **strings and logical operators**.  
3. **Level 3** introduces **loops, functions, random numbers**.  
4. **Level 4** will push your **problem-solving skills**, combining all learned concepts.  
5. Keep a **practice folder** with code files named like `level1_project1.py`.  
6. After finishing, try **combining multiple projects** into bigger apps like a **Banking + Calculator + Password Checker app**.
