# 🐍 Python Practice Exercises (Easy → Medium → Hard)

This practice sheet is designed to strengthen **Python fundamentals** through a gradual progression from **Easy → Medium → Hard** exercises.
It focuses on the following beginner topics:

* Python Basics
* Variables
* Type Casting
* User Input

These exercises are structured so that you can spend **around 1 hour practicing**, while also improving **logic-building and interview readiness**.

---

# 1️⃣ Python Basics

## 🟢 Easy

1. Print `"Hello Python"`.
2. Print your **name**.
3. Print `"I am learning Python"` three times.
4. Print the result of `5 + 10`.
5. Print `"Python is powerful"` on one line.

---

## 🟡 Medium

6. Write a program that prints the **sum of two numbers**.
7. Write a program that prints the **product of two numbers**.
8. Print the result of `20 - 5 * 2`.
9. Write a program that prints the **square of a number**.
10. Write a program that prints the **cube of a number**.

---

## 🔴 Hard

11. Write a program to calculate the **area of a rectangle**.
12. Write a program to calculate the **area of a circle** with radius = 7.

Formula:

```
A = π r²
```

13. Write a program to calculate the **average of three numbers**.
14. Convert **seconds into minutes**.
15. Convert **kilometers into meters**.

---

# 2️⃣ Variables

## 🟢 Easy

1. Create a variable `name` and print it.
2. Create a variable `age` and print it.
3. Create a variable `city` and print it.
4. Store `250` in a variable called `price` and print it.
5. Store `"Learning Python"` in a variable `message` and print it.

---

## 🟡 Medium

6. Create variables `num1` and `num2` and print their **sum**.
7. Create variables `length` and `width` to calculate **rectangle area**.
8. Create variables `a=10`, `b=20`, `c=30` and calculate their **average**.
9. Create a variable `salary` and calculate **annual salary (12 months)**.
10. Store a **temperature value** in a variable and print it.

---

## 🔴 Hard

11. Swap values of `x=10` and `y=20` using a **third variable**.
12. Swap values of `x=10` and `y=20` **without using a third variable**.
13. Calculate **Simple Interest** using variables `P`, `R`, `T`.

Formula:

```
SI = (P × R × T) / 100
```

14. Find the **maximum number among three variables**.
15. Write a program to calculate **total marks of a student**.

---

# 3️⃣ Type Casting

## 🟢 Easy

1. Convert `"25"` into an integer.
2. Convert `50` into a string.
3. Convert `10.75` into an integer.
4. Convert `"3.14"` into a float.
5. Convert `25` into a float.

---

## 🟡 Medium

6. Convert `"10"` and `"20"` into integers and print their **sum**.
7. Convert `"5"` into an integer and print its **square**.
8. Convert `"45.6"` into float and print **double value**.
9. Convert `"100"` into integer and add **50**.
10. Convert `"7"` into integer and print its **cube**.

---

## 🔴 Hard

11. Convert `"25"` into integer and calculate **area of square**.
12. Convert `"8"` into integer and calculate **cube**.
13. Convert `"12.5"` into float and print **half of it**.
14. Convert `"90"` into integer and calculate **percentage of 500**.
15. Convert `"15"` into integer and explore **factorial concept**.

---

# 4️⃣ User Input

## 🟢 Easy

1. Take **name input from the user** and print it.
2. Take **age input** and print it.
3. Take **city input** and print it.
4. Take **favorite color input** and print it.
5. Take **a number input** and print it.

---

## 🟡 Medium

6. Take **two numbers from the user** and print their sum.
7. Take a **number input** and print its **square**.
8. Take **radius input** and calculate **circle area**.
9. Take **length and width input** and calculate **rectangle area**.
10. Take **three numbers** and calculate their **average**.

---

## 🔴 Hard

11. Take a **number** and check whether it is **even or odd**.
12. Take **age input** and check if the user is **adult or minor**.
13. Take **temperature input** and convert it to **Fahrenheit**.
14. Take **marks input** and calculate **percentage**.
15. Take **two numbers** and print the **larger number**.
